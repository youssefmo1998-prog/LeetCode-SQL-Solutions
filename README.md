## 🛠️ التقنيات والمفاهيم المستخدمة (Skills Covered)

المسائل داخل المستودع تغطي جوانب متعددة ومتقدمة في الـ SQL تشمل:

* **Basics & Joins:** الربط بين الجداول بمختلف أنواعه واسترجاع البيانات بكفاءة.
* **Aggregations & Grouping:** عمليات التجميع وحساب الإحصائيات باستخدام `GROUP BY` و `HAVING`.
* **Advanced Subqueries & CTEs:** كتابة الاستعلامات الفرعية والتعبيرات الجدولية الشائعة المعقدة لتنظيم الكود.
* **Window Functions:** الدوال التحليلية والمتقدمة مثل `DENSE_RANK`, `ROW_NUMBER`, و `LEAD/LAG`.
* **Conditional Logic:** استخدام عبارات `CASE WHEN` للتحكم في تدفق البيانات وعرضها.
